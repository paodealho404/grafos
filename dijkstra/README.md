# Algoritmo de Dijkstra

## Compilando
```bash
    cd dijkstra
    make build
```

## Executando

```bash
    ./dijkstra -f entrada.in -o saida.txt -i 1 -l 2
```

## Exemplo de entrada

```bash
    4 6
    1 2 3
    2 1 1
    3 1
    1 3
    3 2
    2 4 5
```

## Saída resultante
>### O custo do menor caminho do vértice s para o vértice t
```bash
    (1, 2): 2
```
